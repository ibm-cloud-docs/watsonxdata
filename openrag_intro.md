# Introduction to OpenRAG (Private preview)
{: #openrag_intro}

OpenRAG is an advanced capability in {{site.data.keyword.lakehouse_full}} that transforms unstructured data into actionable context for AI applications through retrieval-augmented generation (RAG) technology.
{: shortdesc}

## What is OpenRAG?
{: #openrag_what}

OpenRAG combines the power of OpenSearch with retrieval-augmented generation (RAG) to enable AI-powered enterprise search and analytics. It allows you to unlock insights from unstructured data sources such as documents, logs, and text files by converting them into searchable, AI-ready context.

With OpenRAG, you can:

* Transform unstructured data into structured, searchable formats
* Enable semantic search across large volumes of documents
* Provide relevant context to large language models (LLMs) for more accurate AI responses
* Build intelligent search applications that understand natural language queries
* Enhance AI applications with enterprise-specific knowledge

## Key features
{: #openrag_features}

OpenRAG in {{site.data.keyword.lakehouse_short}} provides the following capabilities:

Semantic search
:   Search based on meaning rather than exact keyword matches, enabling more intuitive queries.

Vector embeddings
:   Convert text into numerical representations that capture semantic relationships.

Scalable indexing
:   Handle large volumes of unstructured data with OpenSearch's distributed architecture.

AI integration
:   Seamlessly integrate with watsonx.ai and other LLM platforms for enhanced AI applications.

Real-time updates
:   Keep your search indexes current with real-time data ingestion and updates.

Enterprise security
:   Maintain data security and access controls while enabling AI-powered search.
