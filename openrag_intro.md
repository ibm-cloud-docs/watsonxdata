# Introduction to OpenRAG (Private preview)
{: #openrag_intro}

OpenRAG is an advanced capability in {{site.data.keyword.lakehouse_full}} that transforms unstructured data into actionable context for AI applications through retrieval-augmented generation (RAG) technology.
{: shortdesc}

## What is OpenRAG?
{: #openrag_what}

OpenRAG is an enterprise-grade Agentic RAG framework, available as a fully managed service on watsonx.data SaaS. It combines the power of OpenSearch with retrieval-augmented generation (RAG) to enable AI-powered enterprise search. OpenRAG allows you to unlock insights from unstructured data sources such as documents, logs, and text files by converting them into searchable, AI-ready context.

## Key features
{: #openrag_features}

OpenRAG in watsonx.data provides the following capabilities:

- **Faster time-to-value**: Transform unstructured data into structured, searchable formats.
- **Higher retrieval quality**: Enable semantic search across large volumes of documents.
- **More accurate AI responses**: Provide relevant context to large language models (LLMs) for more accurate AI responses.
- **Natural-language search experiences**: Build intelligent search applications that understand natural-language queries.
- **Enterprise-grade knowledge access**: Enhance AI applications with enterprise-specific knowledge.
